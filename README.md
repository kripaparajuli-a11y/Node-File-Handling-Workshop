# Node.js File Handling Workshop

## Description

This project demonstrates how to perform file handling in Node.js using the built-in File System (fs) module. It includes examples of reading, writing, appending, renaming, deleting files, synchronous and asynchronous operations, and an HTTP server that serves an HTML page.

## Features

- Read files
- Write files
- Append files
- Rename files
- Delete files
- Synchronous file handling
- Asynchronous file handling
- HTTP server with HTML
- Beginner-friendly examples

## Technologies Used

- Node.js
- JavaScript
- HTTP Module
- File System (fs)
- Git
- GitHub

## Installation

Clone the repository:

```bash
git clone https://github.com/kripaparajuli-a11y/Node-File-Handling-Workshop.git
```

Move into the project folder:

```bash
cd Node-File-Handeling-Workshop
```

Install dependencies (if any):

```bash
npm install
```

## Usage

Run an example:

```bash
node readFile.js
```

Run the HTTP server:

```bash
node server.js
```

Open your browser and visit:

```
http://localhost:3000
```
